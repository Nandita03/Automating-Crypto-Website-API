# Automating-Crypto-Website-API

In this project, I aimed to automate the process of pulling data from a cryptocurrency website API using Python programming language. I utilized Python's requests library to send HTTP requests to the API and retrieve the data in JSON format. Moreover, I also cleaned up the data using Python's pandas library to prepare it for analysis. To provide insights into the data, I also incorporated data visualization techniques.
